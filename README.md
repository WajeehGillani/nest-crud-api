## Description
Build CRUD REST APIs with Nestjs Framework and MongoDB database system. Made this application is only for learning purpose.

## Installation

```bash
$ npm install
```

## Running the app
Change the MongoDB URI `mongodb://localhost/nest` with your URI from app.module.ts

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
